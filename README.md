# Assessment of non-protonated perfluorocarbon liquid-filled padding on cervical and lumbar spinal fMRI

## Contents

Full detail of the pipeline are availble in the above manuscript. Briefly, the code is a series of bash and Python scripts using tools from FSL, SCT, AFNI, and Nipype to preprocess and extract image quality metrics for the assessment of non-protonated perfluorocarbon liquid-filled padding (SatPad) on cervical and lumbar spinal fMRI.
